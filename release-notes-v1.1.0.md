## 更新内容

`Start Your Python v1.1.0` 重点把“像 IDE 的学习器”往“能真实运行 Python 示例的教学桌面应用”推进了一步。

### 本次改进
- 右上角主 `Run` 现在只负责进入课程内容态，不再直接执行整份伪代码
- 课程内容里的代码块新增独立 `Run`，点击后会在底部终端真实执行对应示例
- 终端现在接入系统 Python，支持标准输出、错误输出和 `input()` 交互
- 没检测到 Python 时，会在终端里提示，并支持直接跳转到安装 Python 课程后重新检测
- 课程步骤、运行面板和真实终端的职责重新拆开，交互更接近教学版 PyCharm

### 使用说明
- 本版本依赖本机已安装的 Python 解释器
- 课程内容代码块运行时，会优先调用系统 `python` 或 `py`
- 如果本机没有可用 Python，可以先学习安装课程，再回到终端重新检测

### 桌面版本
- 当前提供 Windows x64 桌面包
- 下载后解压，运行 `StartYourPython.exe` 即可使用
